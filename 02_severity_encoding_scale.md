# 0.3.3 Severity Encoding Scale

## Purpose
This document defines what the 0-10 severity scale means clinically for Symtra.

## Severity Scale

| Score Range | Meaning | Typical Interpretation |
|---|---|---|
| 0 | none | symptom absent |
| 1-2 | minimal | noticeable but trivial |
| 3-4 | mild | uncomfortable but daily activities preserved |
| 5-6 | moderate | clearly impactful, may reduce normal activity |
| 7-8 | severe | difficult to perform normal daily tasks |
| 9 | very severe | nearly unbearable, major functional impairment |
| 10 | extreme | worst imaginable / intolerable / emergency concern |

## Practical Guidance
- 1-2: barely noticeable, mild nuisance
- 3-4: mild but real symptom
- 5-6: patient is affected and may change behavior due to symptom
- 7-8: symptom strongly limits activity and should increase triage level
- 9-10: strong emergency concern, especially with dangerous symptom type

## Safety Rules
- Pain >= 7/10 should raise concern at least to SEEK_HELP_SOON
- Pain >= 9/10 should strongly suggest SEEK_HELP_IMMEDIATELY when combined with focal pain, chest pain, neuro symptoms, or abdominal danger pattern
- Breathing difficulty should not rely only on numeric severity; dangerous descriptors override scale
